## 正文

```
天天用 Claude Code 的姐妹们🙋 有没有这种崩溃瞬间——
明明记得上周让它调过一个 bug / 画过一块板子，
结果 `claude --resume` 只给一堆看不懂的标题，翻到手酸都找不到😭

我给自己写了个小工具 ccsearch，真的救命👇

✨ 全文搜索：搜的是你当时说过的话、跑过的命令、改过的文件，不只是标题
🏆 智能排序：BM25 算法，最相关的会话自动顶上来
↩️ 一键恢复：选一个，终端直接跳回那个项目，继续聊
🀄 中文超友好：两个字也能搜，比如「检索」
🔒 100% 本地：会话不上传，隐私放心
⚡ 单文件、超快，热搜索几十毫秒

用起来就一句话：
ccs 你还记得的关键词 → 选个数字 → 回到现场✅

装起来也简单：cargo install ccsearch
开源免费，github 搜 ccsearch 就有～

有同样困扰的姐妹冲！评论区聊聊你都用 CC 干过啥👇

#ClaudeCode #程序员 #效率工具 #开发者 #命令行 #Rust #AI编程 #独立开发 #开源项目 #程序员日常
```
