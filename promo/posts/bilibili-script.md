# 哔哩哔哩 (Bilibili)

**说明：** OpenCLI 的 bilibili 适配器只支持 follow / favorite，**不支持投稿上传**。所以这里给的是一条 60–90 秒短视频的脚本 + 稿件信息，录屏后用 B 站创作中心（或手机端）投稿。视频形式最适合展示「搜索 → 恢复」的动态过程。

---

## 稿件标题

```
用 Claude Code 老是找不到之前的会话？一行命令搞定
```

## 简介（含链接）

```
claude --resume 只给标题，根本找不到想要的会话。ccsearch 对整个历史做全文搜索 +
BM25 排序，一键跳回任意会话。Rust 单文件、中文友好、100% 本地、开源免费。
项目地址：https://github.com/yaalsn/ccsearch
安装：cargo install ccsearch
```

## 60–90 秒脚本（口播 + 画面）

**[0–8s 痛点钩子]**
> 口播：「天天用 Claude Code，但想找回上周那个会话，`--resume` 只给你一堆看不懂的标题——是不是很崩溃？」
> 画面：录屏演示 `claude --resume` 一长串标题、来回翻找。

**[8–20s 一句话方案]**
> 口播：「这个工具叫 ccsearch，命令是 ccs。它搜的不是标题，是你当时说过的每一句话、跑过的每条命令。」
> 画面：终端敲下 `ccs logo pcb silkscreen`，出现排好序的结果 + 高亮片段。

**[20–35s 核心爽点：恢复]**
> 口播：「选一个数字，终端直接 cd 回那个项目目录，帮你 `claude --resume`，退出也留在那儿。」
> 画面：输入数字，光标跳进项目目录，会话恢复。

**[35–50s 加分项]**
> 口播：「中文也能搜，两个字都行；BM25 排序把最相关的顶上来；单个二进制、超快、100% 本地，不上传你的会话。」
> 画面：演示 `ccs 全文 检索`；快速闪过「local / <50ms / single binary」字幕。

**[50–65s 收尾 CTA]**
> 口播：「一行 `cargo install ccsearch` 就能用，开源免费，链接放简介了。觉得有用的话，一键三连，github 点个 star～」
> 画面：`cargo install ccsearch`，结尾放仓库地址大字卡 + 三连引导。
