If you use Claude Code daily, you've probably lost a conversation you know you
had. The built-in resume only lists AI-generated titles — so if you don't
recognize the title, you're scrolling.

I built ccsearch to fix that.

It indexes the full text of your entire Claude Code history — prompts, replies,
the commands it ran, the files it touched, even subagent transcripts — and ranks
matches with BM25. You type what you remember, pick a result, and your terminal
jumps straight back into that project with the conversation restored.

A few things I care about in developer tools, and built in here:

• Fast — a single static Rust binary, sub-50ms warm search, no runtime.
• Private — 100% local; your sessions never leave your machine.
• Inclusive — Chinese/CJK-aware, so 2-character queries work where standard
  full-text search can't.

It's open source (MIT). If you're a Claude Code user, I'd genuinely value your
feedback on whether it surfaces the right session for you.

Link in the comments. 👇
