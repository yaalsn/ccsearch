**标题：**
```
ccsearch：全文搜索 + 一键恢复你的 Claude Code 历史会话（Rust 单文件）
```

**正文：**
```
每天用 Claude Code，最烦的一点是 `claude --resume` 只给你一列 AI 生成的标题。
标题要是没认出来，就只能一直翻——明明记得上周调过那个 flaky test、画过那块 PCB，
就是找不到是哪个会话。

于是写了个小工具 ccsearch（命令是 ccs）：把每个会话的全文都索引进去——你的提问、
Claude 的回复、它跑过的命令、动过的文件，甚至 subagent 的记录——用 BM25 排序。
输入你还记得的几个词，选一个结果，终端就直接 cd 回那个项目目录并 `claude --resume`
恢复会话，退出后也留在那个目录。

几个点：
- 中文友好。FTS5 自带分词做不了中文子串搜索（trigram 要 ≥3 字，unicode61 把一整串
  汉字当一个 token），所以自己写了个分析器：中文按重叠二元组切（全文检索 → 全文 文检
  检索），两个字的查询也能命中；英文保持整词 + 前缀。
- Rust 单文件二进制，SQLite/FTS5 直接编进去，无运行时依赖；热搜索 <50ms。
- 100% 本地，会话不出本机（只有用可选的语义模式 -e 时才会把「查询词」发出去）。

MIT 开源：https://github.com/yaalsn/ccsearch
欢迎拍砖，尤其是排序和中文分词这块。
```
